  1. VTT Enhancement Suite확장을 설치합니다.

  4. VTT가 활성화된 Roll20 페이지에서 '핸드아웃' 탭을 엽니다.
  5. VTTES Importer - Import Characters & Handouts 버튼을 눌러 2번에서 저장한 JSON 파일들을 업로드합니다.
