Roll20 CoC 캐릭터 시트

Roll20 기반 Call of Cthulhu 7th 시트를 작성, 백업, 저장, Roll20으로 불러올 수 있습니다.

저장: 선택한 슬롯에 저장합니다.

불러오기: 선택한 슬롯을 현재 페이지에 불러옵니다.

삭제: 선택한 슬롯의 데이터를 삭제합니다.

데이터는 브라우저에 저장되므로 브라우저를 초기화하면 데이터가 삭제됩니다.
다른 브라우저 간에는 데이터를 공유할 수 없습니다. 데이터를 공유하려면 아래 내보내기를 이용하세요.

'Roll20 VTT 내보내기' 혹은 'Roll20 VTT-export'로 다운받은 JSON 파일을 현재 시트에 불러옵니다.

불러오기를 클릭하면 편집중이던 내용을 덮어씁니다.

Roll20 VTT?　　Roll20 사용을 도와주는 비공식 브라우저 확장프로그램
클릭하면 다운로드 페이지로 이동합니다.

Roll20 VTT로 내보내기: 시트의 데이터를 Roll20 VTT 형식으로 내보냅니다.
Roll20 VTT로 내보낸 파일을 Roll20에 업로드하면 시트를 불러올 수 있습니다.
